## Security Philosophy
As a cybersecurity student and an artist, I approach information security with a philosophy grounded in the **CIA Triad: Confidentiality, Integrity, and Availability**. I view the protection of data not only as a technical responsibility but also as an ethical and thoughtful practice. **Confidentiality** ensures that sensitive information is accessible only to authorized individuals, **Integrity** guarantees that data remains accurate and unaltered, and **Availability** ensures that systems and information remain reliably accessible when required. By intentionally mapping my learning and lab work to these principles, I strive to create secure systems that are both dependable and resilient, reflecting the integration of technical precision and creative problem-solving that my artistic perspective brings to cybersecurity.

## Lab Infrastructure Explanation
Within my cybersecurity lab, I employ isolated environments to explore network and system security concepts. Utilizing private or **host-only network adapters** ensures that virtual machines (VMs) remain contained, preventing any impact on my host system or external networks. This lab environment allows me to simulate realistic network topologies, conduct penetration testing, and analyze vulnerabilities in a controlled and safe manner. By thoughtfully designing my lab infrastructure, I can experiment with security tools, monitor network traffic, and evaluate defensive strategies while maintaining a risk-free workspace—an essential consideration as I continue to develop my technical skills in this challenging field.

---

## Virtualization Overview
I rely heavily on **Type 2 Hypervisors**, such as VirtualBox, which operates on top of my host operating system to create and manage virtual machines. These VMs replicate complete computing environments, allowing the installation of different operating systems, configuration of network devices, and simulation of attacks without jeopardizing my primary system. Unlike containers, which share the host operating system and are lightweight, VMs provide full isolation with their own operating system, libraries, and applications. Virtualization enables hands-on learning, safe experimentation, and a flexible platform to practice fundamental cybersecurity principles such as system hardening, patch management, and secure configuration—critical skills for addressing real-world security challenges.
